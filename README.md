# Super Store Sales Dashboard (Sales Forecasting)

## Project Overview
This project is an interactive Super Store Sales Dashboard created using Power BI. The dashboard helps analyze sales performance, profit trends, category-wise sales, region-wise performance, customer segments, and sales forecasting.

## Objective
The main objective of this project is to understand business performance and identify sales trends for better decision-making.

## Tools Used
- Power BI
- Microsoft Excel
- Data Cleaning
- Data Visualization
- Sales Forecasting

## Key Insights
- Sales and profit analysis
- Category-wise and sub-category-wise sales performance
- Region-wise business performance
- Customer segment analysis
- Sales forecasting for future trends

## Skills Demonstrated
- Data Analysis
- Data Visualization
- Dashboard Development
- Power BI Reporting
- Sales Forecasting

## Project Outcome
Created an interactive dashboard that provides clear business insights and helps understand sales performance effectively.
